Demo: WebAssembly + Web Workers + Canvas

This "n-body problem" is CPU intensive, and this demo uses new browser tech to push hardware limits: WebAssembly, AssemblyScript, Web Workers, and Canvas.

Solar system = stars + gas giants + "debris". Here is a barely stable star and 3 gas giants. Can you add inner planets, without destabilizing your gas giants?
